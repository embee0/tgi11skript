# Glossar

```{glossary}

Syntax
  Unter _Syntax_ versteht man die Regeln, die festlegen, wie Programme in einer Programmiersprache geschrieben werden müssen - also sozusagen die Rechtschreibung und Grammatik der Programmiersprache.<br>
  Z.B. schreibt die Syntax von Python vor, dass if-Anweisungen mit einem Doppelpunkt enden und der nachfolgende Code eingerückt sein muss.<br>
  Im Gegensatz zur {term}`Semantik` geht es bei der Syntax also "nur" um die Form, nicht um den Inhalt.

Semantik
  Die _Semantik_ einer Programmiersprache beschreibt, was die einzelnen Befehle und Anweisungen bewirken.
  Vgl. {term}`Syntax`.

```
